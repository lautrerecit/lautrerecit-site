Ce que le mot « intégration » nous empêche de voir

On présente l'intégration comme un point d'arrivée : l'autre se rapproche, s'efface dans le commun, et le problème disparaît. Mais ce vocabulaire fait un travail discret. Il désigne toujours celui qui doit bouger — jamais la structure qui décide du seuil.

Stuart Hall l'avait nommé : représenter, c'est déjà gouverner. Tant que le récit dominant garde la main sur les mots, il garde la main sur ce qu'on peut penser.

Allah ne modifie pas l'état d'un peuple tant que les individus qui le composent ne modifient pas ce qui est en eux-mêmes.Ar-Ra'd · 13:11

C'est tout l'enjeu : diagnostiquer la domination sans s'y arrêter. Le reste de l'analyse est en ligne.

Note technique

De la maquette à l'e-mail réel

Ces maquettes sont en CSS moderne (pour les voir nettes ici). L'e-mail envoyé se code autrement, sous peine de casser dans Outlook / Gmail :

  • Mise en page en tableaux + CSS inline, largeur 600 px.
  • Polices web non garanties → repli Georgia (Playfair) et système (League Spartan). Le wordmark doré peut devenir une image PNG pour rester fidèle partout.
  • Les dégradés dorés tombent en aplat #C4A24A sur Outlook → prévoir la couleur de secours.
  • Toujours un vrai texte de pré-en-tête (la ligne grise dans la boîte de réception) + lien de désabonnement.

Dis-moi la direction retenue → je la livre en HTML d'envoi testé, prêt à coller dans l'outil.
